## Setup

### Requirements

- .NET 8

### Install

.NET: https://learn.microsoft.com/en-ca/dotnet/core/install/linux-scripted-manual#scripted-install (latest)

Add dotnet cli to path: `export PATH="$PATH:$HOME/.dotnet"`

## Run Unit Tests
